# Computational Modeling of Cognition and Behavior - Chapters

This directory contains the book split into individual chapter files for easier navigation and reading.

## Book Information

**Title:** Computational Modeling of Cognition and Behavior  
**Authors:** Simon Farrell and Stephan Lewandowsky  
**Publisher:** Cambridge University Press  
**Year:** 2018

## File Structure

### Front Matter
- `00_Front_Matter.md` - Title page, Contents, Tables, Contributors, Preface

### Part I: Introduction to Modeling
- `Chapter_01_Introduction.md` - Models and Theories, Quantitative Modeling, Scope and Falsifiability
- `Chapter_02_From_Words_to_Models_Building_a_Toolkit.md` - Building simulations, Random-Walk Model

### Part II: Parameter Estimation
- `Chapter_03_Basic_Parameter_Estimation_Techniques.md` - Discrepancy functions, Least-squares estimation
- `Chapter_04_Maximum_Likelihood_Parameter_Estimation.md` - Likelihoods, probability distributions, ML properties
- `Chapter_05_Combining_Information_from_Multiple_Participants.md` - Averaging, Vincentizing, individual fitting
- `Chapter_06_Bayesian_Parameter_Estimation.md` - Basic concepts, analytic methods, priors
- `Chapter_07_Bayesian_Parameter_Estimation.md` - Monte Carlo methods, MCMC, ABC
- `Chapter_08_Bayesian_Parameter_Estimation.md` - JAGS language and implementation
- `Chapter_09_Multilevel_or_Hierarchical_Modeling.md` - Hierarchical models, Bayesian and ML approaches

### Part III: Model Comparison
- `Chapter_10_Model_Comparison.md` - Goodness of fit, complexity, AIC, BIC
- `Chapter_11_Bayesian_Model_Comparison_Using_Bayes_Factors.md` - Marginal likelihoods, Bayes factors

### Part IV: Models in Psychology
- `Chapter_12_Using_Models_in_Psychology.md` - Modeling workflow, drawing conclusions, best practices
- `Chapter_13_Neural_Network_Models.md` - Hebbian learning, backpropagation
- `Chapter_14_Models_of_Choice_Response_Time.md` - Sequential sampling models, diffusion model, LBA
- `Chapter_15_Models_in_Neuroscience.md` - Neural correlates, reinforcement learning

### Appendices and References
- `Appendix_Appendix_A_Greek_Symbols.md` - Greek symbol reference
- `Appendix_Appendix_B_Mathematical_Terminology.md` - Mathematical terms
- `References.md` - Complete bibliography
- `Index.md` - Subject index
